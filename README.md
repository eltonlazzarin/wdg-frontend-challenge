## :information_source: How To Use

```bash
# Clone this repository
$ git clone https://github.com/eltonlazzarin/wdg-frontend-challenge.git

# Go into the repository
$ cd wdg-frontend-challenge

# Install dependencies
$ yarn

# Run the app
$ yarn start
```
